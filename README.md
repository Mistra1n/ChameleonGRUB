3. Usage Examples
bash

# Install with 30+ preloaded themes
sudo ./install.sh

# Preview next theme
sudo shuffle-grub-theme --preview

# Download new theme
sudo shuffle-grub-theme --download https://gnome-look.org/s/1234

# List themes
sudo shuffle-grub-theme --list
